# Rubric: engagement-quality

**What it measures:** Whether players come back and share — retention signal.

## Scoring Guide (Phase 2 only)

| Score | Meaning |
|-------|---------|
| 0.0–0.3 | No retention data |
| 0.3–0.5 | Players sign up but don't return for second match day |
| 0.5–0.7 | D1 retention >40% (players return for next match) |
| 0.7–0.9 | D1 >60% and organic shares happening without prompting |
| 0.9–1.0 | D1 >70%, friend invites happening, community discussing results |

## Key Questions
- What % of players return for the next match day?
- Are players sharing results without being asked?
- Are players inviting friends?
