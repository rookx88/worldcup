# Rubric: concept-uniqueness

**What it measures:** Is the game mechanic genuinely different from DraftKings, ESPN Fantasy, FIFA Official, and Sorare?

## Scoring Guide

| Score | Meaning |
|-------|---------|
| 0.0–0.3 | No concept defined, or concept is a direct clone of existing product |
| 0.3–0.5 | Concept exists but could be described as "X but for World Cup" with no novel mechanic |
| 0.5–0.7 | Has a novel element but core loop is similar to existing DFS products |
| 0.7–0.9 | Clearly different mechanic that existing products don't offer; target audience distinct |
| 0.9–1.0 | Fully differentiated: unique mechanic, unique scoring, unique target player — couldn't be confused with anything else |

## Key Questions
- Can you name a product that does exactly what Copa is proposing? (Lower = better)
- Would a casual fan who bounced from DraftKings find this different enough to try?
- Is the unique hook explainable in one sentence?
