# Rubric: user-acquisition

**What it measures:** Actual traction — real people signing up, joining, or playing.

## Scoring Guide (Phase 2 only)

| Score | Meaning |
|-------|---------|
| 0.0–0.2 | 0 signups |
| 0.2–0.4 | 1–25 signups |
| 0.4–0.6 | 26–100 signups |
| 0.6–0.8 | 101–500 signups |
| 0.8–0.9 | 501–999 signups |
| 0.9–1.0 | 1,000+ signups or active players |

## Key Questions
- What is the current waitlist number?
- What channel drove the most signups?
- What is the conversion rate from visit to signup?
