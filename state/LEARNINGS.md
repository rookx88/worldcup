# Learnings

## What Has Already Failed (Do Not Revisit)
- **Prediction Cards** (Gen 1-5, first run): Players pick match outcomes before kickoff and get a shareable card. Rejected — this is a prediction game, dozens exist.
- **Copa Calls / Live Instinct Calls** (Gen 1-10, second run): Binary yes/no calls during the match, shareable card after. Rejected — still prediction-adjacent. The timing is different but the mechanic is the same category.

Copa must not return to either of these concepts in any form, including variations or simplifications.
